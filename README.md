# Wirawisata

## How to clone this repository

```bash
git clone github.com/panntod/Wirawisata-Frontend.git
```

## Set up, Instalation dependencies

```bash
npm install
```

## How to run this application

```bash
npm run dev
```

After that, open your browser at `localhost:3000`
